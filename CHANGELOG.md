# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-15

### Added
- Initial release
- User authentication with JWT
- Real-time check-in/check-out system
- Attendance dashboard with statistics
- User management system (admin functions)
- Attendance reporting and export
- Role-based access control
- Responsive UI with modern design
- PostgreSQL database schema
- Docker and Docker Compose support
- Comprehensive API documentation
- Development and installation guides

### Features
- ✅ Login/Register functionality
- ✅ Check-in and Check-out recording
- ✅ Attendance statistics dashboard
- ✅ User management interface
- ✅ Reports and data export
- ✅ Mobile-responsive design
